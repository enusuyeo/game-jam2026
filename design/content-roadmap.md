# Content Roadmap (MVP)

## Milestone Plan

### M0 - 설계 고정
- 본 문서군 확정
- Open Decisions 우선순위 결정
- 데이터 스키마 초안 확정

### M1 - 1층 세로 슬라이스
- 카드 20장, 적 6종, 보스 1종
- Floor 1 전체 루프(전투 4/선택지 4/보스 1/덱 재구성 1) 플레이 가능
- 2D 전투 배경/보스 스프라이트 파이프라인 확정
- 엔딩 판정 로직 연결

### M2 - 중반층 확장
- 카드 40장, 적 12종, 보스 3종
- Floor 1~3 플레이 가능
- 선택지 이벤트 12개 이상 추가
- 보스 컷씬 6개(Floor 1~3 intro/defeat) 구현

### M3 - 5층 루프 완성
- 보스 5종 확정
- Floor 1~5 완주 가능
- 층 보상 선택 및 덱 재구성 시스템 고정
- 보스 컷씬 총 10개(Floor 1~5 intro/defeat) 완료
- 엔딩 컷씬 3개(`loop/true/fail`) 완료
- 진엔딩 연출/기억 파편 해금 연결

### M4 - 밸런스/폴리시
- 승률/클리어율 지표 수집
- 층별 난이도 밴드 조정
- 사운드/이펙트/카드 UX polish

## Scope Guard
- MVP 기간에는 3D 전투 연출 제외 (2D 그래픽 전용)
- 카드 텍스트는 기능 우선, 고급 로컬라이징은 후순위
- 온라인 기능/리더보드는 포스트 MVP

## Done Criteria (MVP)
- 일반 엔딩과 진엔딩이 모두 재현 가능
- 5층 완주 가능
- 보스 컷씬 10개 + 엔딩 컷씬 3개 재생 가능
- 최소 40분 플레이 루프 성립
