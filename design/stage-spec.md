# Stage Spec

## Stage Philosophy
- 스테이지는 난이도 상승 구간이면서 내러티브 떡밥 전달 구간이다.
- 각 층은 하나의 테마 기믹을 가진다.
- MVP는 `3스테이지` 대신 `5층 고정 루프`를 기준으로 설계한다.

## Stage Layout (MVP)
- 총 5층 고정
- 각 층은 다음 구성으로 고정:
  - 전투 4회
  - 선택지 4회
  - 보스전 1회
  - 층 보상 선택 및 덱 재구성 1회
- 5층 보스 클리어 시 엔딩 판정으로 진입

## Floor Loop Rule
- 각 층의 전투/선택지 총량은 고정하고, 노드 배치는 시드 기반으로 변형 가능
- 선택지는 이벤트/휴식/상점/리스크 교환 중 1개 타입으로 처리
- 층 보상 선택 및 덱 재구성은 보스전 직후에만 제공

## Map Visibility Rule
- 런 시작 시 맵은 안개 상태이며, 현재 위치 기준 인접 노드만 공개한다.
- 같은 런에서 한 번이라도 진입/확인한 노드는 이후 계속 공개 상태를 유지한다.
- 이전 회차에서 방문 이력이 있는 노드는 새 런 시작 시에도 공개 상태로 유지한다.
- 미공개 노드는 경로만 표시하고, 노드 타입/위험도 정보는 숨긴다.

## Boss Cutscene Rule
- 모든 층 보스는 전투 전 `boss_intro` 컷씬 1개를 가진다.
- 모든 층 보스는 처치 후 `boss_defeat` 컷씬 1개를 가진다.
- 컷씬은 스킵 가능해야 하며, 스킵 시 전투/보상 흐름이 즉시 이어져야 한다.

## Floor Definitions

### Floor 1 - 낙인의 감방
- 목적: 튜토리얼 겸 기초 전략 학습
- 기믹: 스트레스 수치가 빠르게 상승, 방어 카드 가치 강조
- 보스: `철쇄 간수`

### Floor 2 - 균열의 저장고
- 목적: 리스크/리턴 선택 학습
- 기믹: 강화 보상과 패널티 선택 빈도 증가
- 보스: `미정`

### Floor 3 - 망각의 복도
- 목적: 빌드 분기 완성
- 기믹: 적이 버프/디버프 주기를 사용, 해제 카드 중요
- 보스: `기억 집행관`

### Floor 4 - 심판의 현관
- 목적: 덱 압축/정화 판단 검증
- 기믹: 덱 오염 디버프와 제거 선택지 동시 압박
- 보스: `미정`

### Floor 5 - 문지기의 제단
- 목적: 엔딩 판정 직전 최종 검증
- 기믹: 귀속 카운트에 따라 보스 패턴 일부 변화
- 보스: `타르타로스의 문지기`

## Event Design Rule
- 각 선택지 이벤트는 반드시 다음 중 1개 이상 포함:
  - 단기 이득 / 장기 손해
  - 체력 이득 / 귀속 증가
  - 덱 강화 / 즉시 전투 난이도 증가

## Rewards by Node
- 일반 전투: 골드 + 카드 선택
- 선택지: 결과 기반 보상/패널티 (회복/정화/유물/저주/골드 변동)
- 보스전: 유물 또는 핵심 카드군 선택
- 층 보상 선택 및 덱 재구성: 카드 제거/강화/교체 중 1회 이상 수행

## Open Decisions
- Floor 2/4 보스 콘셉트 및 패턴 확정
- 층별 선택지 타입 비율 (이벤트/휴식/상점/리스크)
- 층 보상 선택 UI (1회 고정 vs 복수 액션 포인트 방식)
- 회차 간 방문 이력 저장 단위 (노드 ID vs 좌표 기반)
